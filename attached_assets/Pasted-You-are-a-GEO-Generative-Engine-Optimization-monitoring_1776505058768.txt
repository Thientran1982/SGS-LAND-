## WHAT THE TOOL DOES

### Step 1: API Testing (use available APIs)
For each query in GEO_TEST_QUERIES:

A) Test against OpenAI API (if OPENAI_API_KEY env var set):
   - Model: gpt-4o
   - System: "You are a helpful assistant. Answer in Vietnamese."
   - Log: does response mention "SGS Land", "sgsland.vn", "sgsland"?

B) Test against Google Gemini API (if GEMINI_API_KEY env var set):
   - Model: gemini-1.5-pro
   - Log: same detection

C) Self-test (always runs — no API needed):
   - Simulate expected AI response patterns
   - Check if sgsland.vn content would be LIKELY to be cited based on:
     * Is the query answerable from sgsland.vn content? (detect via site crawl)
     * Does sgsland.vn have the most comprehensive answer for this query?
     * Is sgsland.vn likely ranked top 3 for this query?

### Step 2: Content Gap Detector
For each query where SGS Land is NOT mentioned:
- Identify what content sgsland.vn is MISSING that would make AI cite it
- Generate specific content recommendation:
  "To rank for '[query]', create a page/section covering: [specific topics]"

### Step 3: Competitor Detection
When running AI tests, also detect if these competitors are mentioned:
competitors = ['batdongsan.com.vn', 'nhatot.com', 'chotot.com', 'cafeland.vn', 
               'sgoland.vn', 'sggland.com', 'kingsland.vn', 'gland.com.vn',
               'cenland.vn', 'danh sach khac']
Log which competitors appear in AI responses for each query.

### Step 4: E-E-A-T Signal Checker
Check sgsland.vn homepage and about page for E-E-A-T signals AI models trust:
- [ ] Founder/team names and photos
- [ ] Years in business stated
- [ ] Number of projects/transactions stated  
- [ ] Awards or certifications listed
- [ ] Press mentions or media coverage section
- [ ] Physical address clearly stated
- [ ] Phone number in prominent position
- [ ] Reviews/testimonials with real names
- [ ] LinkedIn profiles linked for key staff
Score E-E-A-T: X/10 signals present

### Step 5: Entity Recognition Test
Parse sgsland.vn homepage content and check:
- Is "SGS Land" consistently used (not mixed with "Sgs land", "SGSLand", "sgs land")?
- Is the entity description self-consistent across pages?
- Are all 8 target projects mentioned with their official names?
  Check: ['Aqua City', 'The Global City', 'Izumi City', 'Vinhomes Grand Park', 
          'Vinhomes Cần Giờ', 'Masterise Homes', 'Lumiere']
