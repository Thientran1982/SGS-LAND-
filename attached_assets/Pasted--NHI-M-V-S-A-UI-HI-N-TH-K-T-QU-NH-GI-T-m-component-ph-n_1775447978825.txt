--- PHẦN 1: HÀM FORMAT TIỀN VIỆT NAM ---

Thêm hàm này, dùng ở mọi nơi hiển thị số tiền:

function formatVND(ty) {
  if (ty >= 1) {
    const rounded = Math.round(ty * 100) / 100;
    return rounded.toFixed(2).replace('.', ',') + ' tỷ';
  } else {
    const trieu = Math.round(ty * 1000);
    return trieu.toLocaleString('vi-VN') + ' triệu';
  }
}
// Ví dụ: formatVND(6.5) → "6,50 tỷ"
// Ví dụ: formatVND(0.85) → "850 triệu"

function getLevel(c) {
  return c >= 80 ? 'high' : c >= 70 ? 'med' : 'low';
}

--- PHẦN 3: BẢNG BREAKDOWN ĐỊNH GIÁ ---

Thêm section "Cơ sở định giá" (accordion, mặc định đóng):

Quy tắc màu sắc cột "Điều chỉnh":

- Âm (-): màu đỏ (#A32D2D)
- Cơ sở: màu xám (#5F5E5A)

--- PHẦN 6: LƯU LỊCH SỬ LOCAL ---

Sau mỗi định giá thành công, lưu vào localStorage:

function saveHistory(inputData, result) {
  const history = JSON.parse(localStorage.getItem('sgs_history') || '[]');
  history.unshift({
    id: Date.now(),
    time: new Date().toLocaleString('vi-VN'),
    dia_chi: inputData.dia_chi,
    loai_hinh: inputData.loai_hinh,
    gia: result.gia_uoc_tinh,
    do_tin_cay: result.do_tin_cay,
  });
  localStorage.setItem('sgs_history', JSON.stringify(history.slice(0, 10)));
}

Thêm link "Lịch sử định giá (X lần)" ở cuối trang.
Click → hiện modal danh sách 10 lần gần nhất dạng bảng.
