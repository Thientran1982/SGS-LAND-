## WHAT TO BUILD

Create a file: seo-geo-audit.mjs

### 1. TECHNICAL SEO CHECKS
Crawl https://sgsland.vn and for each page found:
- [ ] HTTP status code (200, 301, 302, 404, 500)
- [ ] Response time (TTFB in ms) — flag if > 600ms
- [ ] Page title: exists? length 50–60 chars? contains target keyword?
- [ ] Meta description: exists? length 140–160 chars?
- [ ] H1: exactly one? contains keyword?
- [ ] H2–H6: proper hierarchy?
- [ ] Canonical tag: exists? self-referencing or pointing elsewhere?
- [ ] robots meta tag: noindex present? (flag as CRITICAL if yes)
- [ ] Open Graph tags: og:title, og:description, og:image present?
- [ ] Twitter Card tags present?
- [ ] Hreflang tags (check if vi/en versions exist)
- [ ] Broken internal links (4xx)
- [ ] Image alt text: count images missing alt
- [ ] Image size: flag images > 150KB (check Content-Length)
- [ ] Structured data: detect JSON-LD blocks, validate type exists
  - Required types for real estate: Organization, RealEstateListing, FAQPage, BreadcrumbList, LocalBusiness
- [ ] Word count per page (flag if < 300 words for project pages, < 1500 for blog)
- [ ] Keyword density for target terms: ['aqua city', 'global city', 'izumi', 'vinhomes can gio', 'masterise', 'bat dong san', 'sgs land', 'sgsland']

### 2. CORE WEB VITALS (Lighthouse)
Run Lighthouse headless for homepage and 3 key project pages:
- LCP (Largest Contentful Paint) — pass: < 2500ms, warn: < 4000ms, fail: >= 4000ms
- CLS (Cumulative Layout Shift) — pass: < 0.1, warn: < 0.25, fail: >= 0.25
- INP (Interaction to Next Paint) — pass: < 200ms, warn: < 500ms, fail: >= 500ms
- FCP (First Contentful Paint) — pass: < 1800ms
- TTFB — pass: < 600ms
- Performance Score overall
- SEO Score from Lighthouse
- Accessibility Score

### 3. SITEMAP & ROBOTS.TXT AUDIT
- Fetch https://sgsland.vn/sitemap.xml — does it exist?
- Fetch https://sgsland.vn/robots.txt — does it exist?
- Parse sitemap: count URLs, detect missing pages
- Check robots.txt: is Googlebot blocked? Is sitemap declared?
- Check if sitemap is submitted (can't verify directly — flag as manual check)

### 4. SCHEMA / STRUCTURED DATA AUDIT
For each JSON-LD block found:
- Detect @type
- Check required fields per type:
  - Organization: name, url, logo, contactPoint, sameAs
  - LocalBusiness: name, address, telephone, openingHours, geo
  - RealEstateListing: name, description, url, image, offers
  - FAQPage: mainEntity array with Question + acceptedAnswer
  - BreadcrumbList: itemListElement with position, name, item
- Flag missing fields as CRITICAL, WARN, or INFO

### 5. GEO READINESS CHECKS
Check each crawled page for:
- [ ] Author byline present (name + bio snippet) — important for E-E-A-T
- [ ] Publication date + last modified date visible
- [ ] Specific factual data present: numbers, prices, dates, measurements
  - Regex: detect patterns like "từ X tỷ", "X ha", "X căn", "bàn giao năm XXXX"
- [ ] FAQ section present (detect <details>, accordion pattern, or FAQ heading)
- [ ] Self-contained paragraphs: avg sentence per paragraph < 4 (AI-friendly)
- [ ] Citation-worthy claims: detect if prices/specs are sourced or attributed
- [ ] Entity mentions: count mentions of ['Novaland', 'Masterise', 'Vinhomes', 'Vingroup', 'Aqua City', 'Global City', 'Izumi', 'Khu Đông', 'TP.HCM', 'SGS Land']
- [ ] Bing indexing readiness: check for Bing-specific meta tags
- [ ] Google Business Profile signals: check for LocalBusiness schema with geo coordinates
- [ ] Video content: detect YouTube embeds (VideoObject schema check)

### 6. BACKLINK & AUTHORITY SIGNALS (on-page proxies)
Since we can't check external links directly, check:
- [ ] External outbound links to high-authority domains (.gov.vn, known news sites)
- [ ] Social media profile links in footer/header (Facebook, YouTube, LinkedIn, TikTok, X/Twitter)
- [ ] Press/media mention section exists on About page?
- [ ] Awards/certifications section exists?
- [ ] Partner logos section exists?

### 7. MOBILE & ACCESSIBILITY
- [ ] Viewport meta tag present
- [ ] Font size >= 14px (check inline styles and detect <12px)
- [ ] Touch targets >= 44px (flag buttons/links that might be too small)
- [ ] lang attribute on <html> tag

## SCORING SYSTEM

Create a weighted score (0–100) per page:
- Technical SEO: 30 points
- Content Quality: 25 points

- GEO Readiness: 15 points
- Performance: 10 points

Thresholds:
- 90–100: EXCELLENT (green)
- 70–89: GOOD (yellow)
