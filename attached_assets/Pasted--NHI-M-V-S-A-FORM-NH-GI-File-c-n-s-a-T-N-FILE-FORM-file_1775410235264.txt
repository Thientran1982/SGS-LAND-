## NHIỆM VỤ: SỬA FORM ĐỊNH GIÁ

File cần sửa: [TÊN_FILE_FORM] (file form Agent vừa báo cáo ở trên)

--- PHẦN 2: VALIDATION (không dùng thư viện ngoài) ---

Thêm hàm validateForm() vào file form. Dùng JavaScript thuần,
không cài thêm thư viện. Logic:

function validateForm(data) {
  const errors = {};

  if (!data.dia_chi || data.dia_chi.trim().length < 10)
    errors.dia_chi = "Vui lòng nhập địa chỉ đầy đủ (tối thiểu phường, quận, TP)";

  if (!data.dien_tich_dat || data.dien_tich_dat < 15 || data.dien_tich_dat > 50000)
    errors.dien_tich_dat = "Diện tích phải từ 15m² đến 50.000m²";

  if (!data.huong_nha)
    errors.huong_nha = "Vui lòng chọn hướng cửa chính";

  if (!data.vi_tri_duong)
    errors.vi_tri_duong = "Vui lòng chọn vị trí đường";

  if (!data.phap_ly)
    errors.phap_ly = "Vui lòng chọn tình trạng pháp lý";

  if (!data.noi_that)
    errors.noi_that = "Vui lòng chọn tình trạng nội thất";

  if (data.nam_xay_dung) {
    const yr = parseInt(data.nam_xay_dung);
    if (yr < 1975 || yr > 2025)
      errors.nam_xay_dung = "Năm xây dựng phải từ 1975 đến 2025";
  }

  return { isValid: Object.keys(errors).length === 0, errors };
}

Gọi validateForm() khi người dùng submit.
Hiển thị lỗi ngay dưới mỗi field bị sai (màu đỏ, font nhỏ).
Disable nút "Định giá" nếu có lỗi.

--- PHẦN 3: LOADING STATE ---

Khi đang gọi API (isLoading = true):
1. Disable toàn bộ form và nút submit
2. Hiển thị div loading với 4 bước tuần tự:
   - Bước 1 (0s):   "Đang phân tích địa chỉ..."
   - Bước 2 (2s):   "Tìm BĐS tương đồng khu vực..."
   - Bước 3 (4s):   "AI đang tính toán định giá..."
   - Bước 4 (6s):   "Hoàn thiện báo cáo..."
3. Dùng setTimeout để chuyển bước, clearTimeout khi done

--- PHẦN 4: KIỂM TRA SAU KHI SỬA ---

Sau khi sửa xong, chạy app và kiểm tra:
- Form hiển thị đủ 6 trường mới không?
- Trường chiều rộng mặt tiền ẩn/hiện đúng khi chọn vị trí đường không?
- Bấm submit khi thiếu field bắt buộc → có hiện lỗi không?
- Bấm submit khi đủ field → form có submit được không?
- Console có lỗi JS nào không?

Báo cáo kết quả kiểm tra trước khi tiếp tục.
